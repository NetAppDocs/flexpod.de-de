[[cisco-ucs-x-series-direct]]
== Cisco UCS X-Series Direct

Cisco UCS X-Series Direct ist eine neue Architektur, die ein Paar Cisco UCS Fabric Interconnect 9108 100G mit Cisco UCS X9508 Chassis und Cisco UCS X-Series Compute nodes integriert, um die UCS X-Series Direct Lösung bereitzustellen.

Einen schnellen Überblick über die Funktionen und Möglichkeiten der Cisco UCS X-Series bietet link:https://www.cisco.com/c/en/us/products/collateral/servers-unified-computing/ucs-x-series-modular-system/ucs-x-series-direct-aag.html["Cisco UCS X-Series Direct auf einen Blick"^].

Cisco UCS X-Series Direct besteht aus den folgenden Komponenten:

* Cisco UCS 9108 100G Fabric Interconnect
* Cisco UCS X9508 Chassis
* Cisco UCS X-Series Compute-Knoten
* Optionale Komponenten:
+
** PCIe Nodes und X-Fabric Module
** Zweites Cisco UCS X9508 Chassis mit X-Series Compute Nodes und Cisco UCS 9108-100G Intelligent Fabric Modules und bis zu vier C-Series Rack Servern





NOTE: Unter diesem Link sind die vollständigen technischen Spezifikationen für UCS X-Series Direct zu finden: https://www.cisco.com/c/en/us/products/collateral/servers-unified-computing/ucs-x-series-modular-system/ucs-x-series-direct-ds.html["Cisco UCS X-Series Direkt-Datenblatt"].

[[cisco-unified-edge]]
== Cisco Unified Edge

Cisco Unified Edge ist ein KI-fähiges System, das Computing am Edge durch die Zusammenführung von Rechenleistung, Netzwerk, Speicher und Sicherheit neu definiert. Das modulare Design ist zukunftsfähig, energieeffizient und wartungsfreundlich und kann auf heutige Workloads und Anwendungsfälle zugeschnitten werden, bleibt dabei jedoch anpassungsfähig an die sich rasant entwickelnde KI-Landschaft.

Einen schnellen Überblick über die Funktionen und Merkmale von Cisco Unified Edge bietet link:https://www.cisco.com/c/en/us/products/collateral/servers-unified-computing/unified-edge/unified-edge-at-a-glance.html["Cisco Unified Edge auf einen Blick."^]

Cisco Unified Edge besteht aus den folgenden Komponenten:

* Cisco UCS XE9305 Chassis
* Zwei Cisco Unified Edge Management Controller mit integrierten 25G Switches
* Rechenknoten – Auswahl
+
** Cisco UCS XE130c M8 Compute Node
** Cisco UCS XE150c M8 Compute Node
