= FlexPod-Plattformen
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


[role="lead"]
Es gibt zwei FlexPod Plattformen:

* *FlexPod Datacenter.* Diese Plattform ist eine hochskalierbare Rechenzentrumsinfrastruktur, die sich für Enterprise-Workloads und -Anwendungen, Virtualisierung, VDI sowie Public und Private Cloud eignet. FlexPod Datacenter verfügt über eigene Spezifikationen, die in  https://docs.netapp.com/us-en/flexpod/fp-def/dc-tech-spec_solution_overview.html["Technische Spezifikationen für FlexPod Datacenter"^] dokumentiert sind.
* *FlexPod Express.* Diese Plattform ist eine kompakte konvergente Infrastruktur, die sich für Anwendungsfälle in Remote-Zweigstellen und Edge eignet.


Dieses Dokument enthält die technischen Spezifikationen der FlexPod Express Plattform.
