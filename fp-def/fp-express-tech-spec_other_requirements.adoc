= Andere Anforderungen
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


[role="lead"]
Zusätzliche Anforderungen für FlexPod Express sind:

* Wenn VMware ESXi als Hypervisor vorgesehen ist, bietet die link:https://compatibilityguide.broadcom.com/["Broadcom Kompatibilitätsleitfaden"] einen Überblick darüber, ob die Hardware- und Softwarekomponenten unterstützt werden.
* Für eine auf OpenShift basierende Bereitstellung ist sicherzustellen, dass die erforderlichen Hardware- und Softwarekomponenten vorhanden sind, die mit OpenShift kompatibel sind. Weitere Informationen sind im link:https://catalog.redhat.com/en/search["OpenShift Katalog"] zu finden.
