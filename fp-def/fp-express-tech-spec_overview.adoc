= Technische Spezifikationen zu FlexPod Express
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


[role="lead"]
FlexPod Express ist eine vorkonfigurierte, bewährte Architektur, die auf dem Cisco Unified Computing System (Cisco UCS) und der Cisco Nexus-Familie von Switches basiert, und die Speicherschicht wird mit den NetApp AFF-, ASA- oder FAS-Einstiegsspeichersystemen aufgebaut. FlexPod Express ist eine geeignete Plattform für den Betrieb verschiedener Virtualisierungshypervisoren und Bare-Metal-Betriebssysteme (OSs) sowie Unternehmens-Workloads.

FlexPod Express bietet nicht nur eine Basiskonfiguration, sondern auch die Flexibilität, an unterschiedliche Anwendungsfälle und Anforderungen angepasst und optimiert zu werden.
